# Kosen Website Crawler

高専のウェブサイトをクロールしてスクリーンショットを保存します。

## Getting started

required node version >= 7.10.0

```
yarn install
yarn start
```

## How to change crawling sites?

websites.json の内容を変更すると高専のサイト以外も取得することができます。
